
由于美元符用于模版表达式了，所以说如果我们希望在字符串中仅仅表示$这个字符，那么我们需要用到转义：
```
//普通字符串直接使用\$表示
val text = "\$这是美元符"   

//原始字符串要套个娃
val str = """${'$'}这是美元符"""
```


Kotlin中没有那样的三元运算符，只能使用上面的表达式，对于多行代码块的情况，默认最后一行作为返回的结果：

```
//这里判断socre是否大于60，是就得到Yes，否就得到No，并且可以直接赋值给变量 
val res = if (score > 60) "Yes" else "No"

val res = if (score > 60) {
	println("不错啊期末没挂科")
	"Yes"   //代码块默认最后一行作为返回结果
} else {
	println("不会有人Java期末还要挂科吧")
	"No"
}
```

可以自由控制每一轮增加多少，也就是步长
```
for (i in 1..10 step 2) { println(i) }
```

如果我们需要从10到1倒着进行遍历呢？我们可以将`..`替换为`downTo`来使用																																													```
						
```																		fun main() {
    for (i in 10 downTo 1) {
        println(i)   //这里得到的就是10到1倒着排列的范围了
    }
```


break语句，要是我们就是想要终止或者是加速外层循环呢？我们可以为循环语句打上标记：

```
outer@ for (i in 1..3) {   
//在循环语句前，添加 标签@ 来进行标记
        inner@ for (j in 1..3) {
            if (i == j) break@outer  //break后紧跟要结束的循环标记，当i == j时终止外层循环
            println("$i, $j")
        }
    }
```

Unit关键字,在 Java 中，`void` 是一个关键字，代表“什么都没有”。而在 Kotlin 中，`Unit`是一个真实的**单例对象**。这意味着你可以像处理其他变量一样处理它
看到 `: Unit`，你只需要把它理解为：**“这个函数只负责干活，不负责带回结果。”**
```
//本质上应该是返回Unit类型，这个类型表示空，类似于Java中的void，默认情况下可以省略
fun hello(): Unit {  
    println("PHP是世界上最好的语言.kt")
}
```

对于一些内容比较简单的函数，比如上面仅仅是计算两个参数的和，我们可以直接省略掉花括号
```
fun test(a: Int = 6, b: Int = 10) = a + b
```


Elvis 操作符 (`?:`),如果想为可空变量为空时提供一个**默认值**
```
val len = comment?.length ?: 0
```
